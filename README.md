# Data Science Projects

This repository contains data science and machine learning analyses focusing on multivariate statistics and predictive modeling.

## 📂 Projects included

### 1. Decoding Default: Multivariate Analysis of Loan Applicants
A comprehensive analysis focusing on the factors that influence loan default rates. This project uses multivariate statistical techniques to understand the complex relationships between borrower characteristics and loan outcomes.
- **Main Notebook**: `Decoding Default Multivariate Analysis of Loan Applicants.ipynb`

### 2. ValueVision
An analytical notebook exploring additional data patterns and predictive modeling.
- **Main Notebook**: `ValueVision.ipynb`

### 📊 Datasets
- `avocado.csv`: Historical data on avocado prices and sales volume across multiple US markets.

## 🚀 Getting Started

To run these notebooks locally on your machine:

1. **Clone this repository** (if you haven't already):
   ```bash
   git clone <your-repository-url>
   cd <repository-directory>
   ```

2. **Install Dependencies**:
   Ensure you have Python installed along with standard data science libraries. You can typically install the required packages using:
   ```bash
   pip install jupyterlab pandas numpy matplotlib seaborn scikit-learn
   ```

3. **Launch Jupyter**:
   ```bash
   jupyter notebook
   # OR
   jupyter lab
   ```

4. Open the desired `.ipynb` file in your browser to interact with the analysis.
